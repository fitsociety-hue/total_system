## ■ 신규 기능 설계 개요

### ◆ 기능 A — 건수 기반 실적 입력 (COUNT_BASED)

#### 배경
대부분의 프로그램은 출석부(출석체크) 기반으로 실인원/연인원을 집계하지만,
고충처리함(이용인소리함), 민원상담, 자문위원회, 공문처리 등은
**개별 이용인 출석이 아닌 "건수(件數)" 단위**로 실적을 기록합니다.

#### 프로그램 실적 유형 (recordType)
```
"attendance"  → 기존 출석부 방식 (실인원/연인원 자동 집계)
"count"       → 건수 입력 방식 (건수/연인원 직접 입력, 실인원은 선택)
```

#### 건수 입력 필드 구조
```javascript
// localStorage: "sams_count_records"
{
  id: string (UUID),
  programId: string,
  staffId: string,
  date: string (YYYY-MM-DD),
  count: number,          // 건수 (필수)
  people: number | null,  // 실인원 (선택 — 알 수 있는 경우 입력)
  extendedPeople: number | null, // 연인원 (선택)
  note: string,           // 내용 요약 (선택)
  createdAt: string,
}
```

#### 집계 로직
- 실인원: `count_records`에서 `people` 합산 (미입력 시 건수로 대체)
- 건수: `count_records`에서 `count` 합산
- 연인원: `count_records`에서 `extendedPeople` 합산 (미입력 시 건수로 대체)
- 달성률: `건수 / 목표건수 × 100`

## ■ 비밀번호(PIN) 처리 규칙 — 전체 코드에서 반드시 준수

> ⚠️ 0 시작 비밀번호 오류 방지 (필수 구현 사항)

① 비밀번호 필드는 반드시 **String 타입**으로 저장
```js
localStorage: JSON.stringify({...staff, pin: String(pin).padStart(4,"0")})
```
② 입력 필드: `type="password"` + `inputMode="numeric"` + `maxLength={4}`
   → `type="number"` 절대 사용 금지 (leading zero 소실 원인)

③ 입력값 검증: `/^[0-9]{4}$/.test(inputValue)`
   → 숫자로 변환하지 않음

④ 비교 시 String 비교: `storedPin === String(inputPin).padStart(4,"0")`

⑤ 임시 PIN 생성: `String(Math.floor(Math.random()*10000)).padStart(4,"0")`

### 업무실적 자동 집계 로직
1. 선택된 날짜의 `sams_attendance` (출석형) + `sams_count_records` (건수형) 통합
2. 각 프로그램의 일계(당일) / 월계(해당월 누적) / 누계(연간 누적) 자동 산출
3. 목표 대비 달성률 자동 계산
4. **직접 수정 가능**: 자동 집계된 값을 담당자가 수동으로 수정할 수 있음 (수정 시 표시 구분)

### 주요 업무 내용 입력
- 팀원 개인 로그인 시: 본인 이름만 입력 (1개 행)
- 팀장 로그인 시: 팀원 전체 이름 행이 자동 생성됨 (팀원 수 기준)
- 업무 내용 자유 서술 (textarea, 자동 높이 조절)
- 담당자 추가/삭제 가능

## [신규] 일일업무일지 — 전체 적용
### 자동 집계 로직
```javascript
function generateDailyPerformance(staffId, date) {
  const programs = getPrograms(staffId); // 담당 프로그램
  return programs.map(prog => {
    let dayData = { count: 0, people: 0, extended: 0 };
    if (prog.recordType === "attendance") {
      const att = getAttendance(prog.id, date);
      dayData = { count: att.length, people: uniquePeople(att), extended: att.length };
    } else {
      const cr = getCountRecord(prog.id, date);
      dayData = { count: cr?.count || 0, people: cr?.people || 0, extended: cr?.extendedPeople || 0 };
    }
    const monthData = getMonthData(prog.id, getMonth(date));
    const cumData = getCumData(prog.id, getYear(date));
    return {
      programId: prog.id,
      programName: prog.name,
      recordType: prog.recordType,
      targetCount: prog.targetCount,
      day: dayData,
      month: monthData,
      cum: cumData,
      achieveRate: calcRate(cumData.count, prog.targetCount),
    };
  });
}
```

function addCountRecord(data) {

  ]);

function saveDailyLog(data) {
  const ss = SpreadsheetApp.openById(SPREADSHEET_ID);
  let sheet = ss.getSheetByName('업무일지');
  if (!sheet) {
    sheet = ss.insertSheet('업무일지');
    sheet.appendRow(['ID','팀명','직원명','날짜','요일','주요업무',
                     '슈퍼비전','상태','생성일시','수정일시']);
  }
  // 기존 행 덮어쓰기 (같은 staffId + date)
  const data_all = sheet.getDataRange().getValues();
  let targetRow = -1;
  for (let i = 1; i < data_all.length; i++) {
    if (data_all[i][0] === data.id) { targetRow = i + 1; break; }
  }
  const workSummary = (data.workItems||[])
    .map(w => `[${w.personName}] ${w.content}`).join('\n');
  const row = [
    data.id, data.teamName, data.staffName, data.date, data.dayOfWeek,
    workSummary, data.supervision||'',
    data.status, data.createdAt, data.updatedAt
  ];
  if (targetRow > 0) {
    sheet.getRange(targetRow, 1, 1, row.length).setValues([row]);
  } else {
    sheet.appendRow(row);
  }
  return { success: true };
}

// ⑪ 일지 구글 시트 출력 (팀별 통합)
function exportDailyLog(date, teamName) {
  const ss = SpreadsheetApp.openById(SPREADSHEET_ID);
  const timestamp = Utilities.formatDate(new Date(),"Asia/Seoul","yyyyMMdd_HHmm");
  const sheetName = `업무일지_${teamName}_${date}_${timestamp}`;
  let newSheet = ss.getSheetByName(sheetName);
  if (newSheet) ss.deleteSheet(newSheet);
  newSheet = ss.insertSheet(sheetName);
  // 헤더
  newSheet.appendRow([`${teamName} 업무일지`, date]);
  newSheet.appendRow(['구분','사업명','목표건수','일계','월계','누계','달성률(%)']);
  // (실적 데이터는 프론트에서 data로 전달받아 appendRow)
  const url = `https://docs.google.com/spreadsheets/d/${SPREADSHEET_ID}/edit#gid=${newSheet.getSheetId()}`;
  return { success: true, url, sheetName };
}
